# Project context for code review

## Architecture
- Static Next.js export (`output: "export"`) — no server-side rendering, no API routes
- Deployed to GitHub Pages under `/BijdehandApp` basePath in production

## Key files
- `app/page.tsx` — page composition only, no logic
- `components/` — all UI components; `components/ui/` for primitives
- `app/content/` — static JSON data (reviews, FAQ); generated by scripts, not edited manually
- `scripts/` — Node.js scripts that run locally; may use `.env.local`

## Sensitive data rules
- `ASC_KEY_ID`, `ASC_ISSUER_ID`, `ASC_PRIVATE_KEY` — only in `.env.local`, only read in `scripts/`
- `app/content/reviews.json` — public data, safe to commit

## Dependency decisions
- Carousel/animation: Framer Motion (already installed) — do not add Swiper, Embla, etc.
- Icons: lucide-react + @tabler/icons-react — do not add other icon libraries
- Utilities: `clsx`, `tailwind-merge` available via `lib/utils.ts`
